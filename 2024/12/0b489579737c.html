视频课程：Berkeley CS 61A scheme版

教材：《计算机程序的构造和解释》(sicp)，网页版

学习流程：（一周两次）对应教材sicp学习->两集视频->ob视频笔记->ob教材笔记

官方资源：前 CS61A：计算机程序的结构和解释

一个有意思的网站

实操作业环境：(Scheme版本众多，有MIT版，伯克利版甚至在线版等，Racket版是用最简单)

  1. 下载安装Racket
  2. 对应目录下cmd中运行：racket -i -p neil/sicp -l xrepl
    (参考)
  3. UBC的一些定义
    , for those trying to get the obj.scm working in Racket for week 7 and above, you can use obj.rtk from https://github.com/zackads/sicp/tree/main/inst.eecs.berkeley.edu/~cs61as/fa11/lib.
    you might need to do this obj.rtk
    ;[() commented out and replaced cause don’t always need instance variables - Babz
    ; ‘()]
    [()
    (values ‘() ‘())] ; Return empty lists if no instance-vars clause is provided
    otherwise some of the demos don’t work because it will give errors that you didn’t provde enough arguments when you make classes without instance variables
    It took a lot to get it working in 2024 due to the fact that the university made a lot of thier stuff campus only (locked behind a uni login) but I finally got it working. I’ll eventually make a repo with instructions and all the files once I’m done
